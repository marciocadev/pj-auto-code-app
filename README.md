Learn how can I use Projen to optimize code generation

## Project startup
* `mkdir pj-auto-code-app && cd pj-auto-code-app`
* `projen new awscdk-app-ts --cdk-version 2.3.0 --projenrc-ts`
* `yarn test`
* `yarn build`
* `yarn release`
